TimeSeriesBasedCalculationExecution

class TimeSeriesBasedCalculationExecution : CalculationExecution

CalculationExecution implementation based on TimeSeries. TimeSeries values are read from TimeSeriesRepository before the call and the results are saved back with repository after the call. CalculationExecution uses ScriptSynchronizer to wait for synchronization to end in case synchronization is ongoing.
